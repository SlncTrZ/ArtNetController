## 🌟 **What's New in LTS 1.0.0**

### 🎬 **Binary DMX Recording & Playback**
- **Ultra-efficient storage** - 12x smaller than JSON format
- **Real-time playback** - Smooth 60+ FPS on Raspberry Pi
- **CRC validation** - Error detection and correction
- **Multi-threaded I/O** - Buffer management for stable playback

### 🌈 **Rainbow Effects Engine**
- **6-Universe support** - Synchronized color effects across universes 0-5
- **Built-in rainbow show** - 3-minute demo with breathing and wave effects
- **HSV color cycling** - Smooth color transitions
- **Cross-universe waves** - Mesmerizing pattern flows

### 🎵 **Advanced Show Management**
- **Binary + Legacy support** - Backward compatible with old shows
- **Playlist control** - Spotify-style interface with loop/shuffle
- **Real-time progress** - Time display and progress tracking
- **Auto-format detection** - Seamless switching between formats

## 🚀 **Quick Start**

### **Option 1: Download Release (Recommended)**
```bash
# Download from GitHub Releases
https://github.com/truongcongdinh97/DMX-Master/releases/tag/v1.0.0

# Extract and run
DMX-Master-LTS-1.0.0.exe  # Windows
./dmx-master              # Linux
```

### **Option 2: Build from Source**
```bash
# Clone repository
git clone https://github.com/truongcongdinh97/DMX-Master.git
cd DMX-Master

# Install dependencies
pip install -r requirements.txt

# Run application
python main.py
```

### **Quick Demo**
1. **Launch** DMX Master LTS 1.0.0
2. **Go to Show Manager** tab
3. **Select** "Default_Rainbow_Show"
4. **Click** "Play Single" button
5. **Watch** 3 minutes of beautiful rainbow effects! 🌈
