BƯỚC 1: LẤY DEVICE ID
----------------------
1. Mở phần mềm ArtNet Controller
2. Menu: Help  License Activation
3. Copy Device ID (mã 64 ký tự)
   Ví dụ: abc123def456...

BƯỚC 2: GỬI CHO ADMIN
----------------------
Email: truongcongdinh97@gmail.com
Nội dung: "Xin license cho Device ID: abc123..."

BƯỚC 3: NHẬN LICENSE JSON
--------------------------
Admin sẽ gửi cho bạn 1 file JSON

BƯỚC 4: KÍCH HOẠT
------------------
1. Mở phần mềm
2. Help  License Activation
3. Paste toàn bộ JSON vào ô "Step 2"
4. Click "✅ Activate License"
5. XONG!
