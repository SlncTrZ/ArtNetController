## 🎯 **Depence Integration Ready**

### **Setup Instructions:**

1. **In Depence:**
   - Settings → Timecode → Enable "Send Net-timecode"
   - IP: 127.0.0.1, Port: 3040, Format: 25fps

2. **In DMX Master:**
   - Record Tab → Enable "Wait for Timecode Signal"
   - Source: "Net-timecode (Network) - 25fps"
   - Click RECORD → Wait for timecode → Auto-start! 🎉
