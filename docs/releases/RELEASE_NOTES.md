## 🚀 **Quick Start Guide**

### **Option 1: Windows Installer (Easiest)**
1. Download `DMX-Master-LTS-1.0.0-Setup.exe`
2. Run installer as Administrator
3. Follow installation wizard
4. Launch from Start Menu or Desktop
5. Play "Default_Rainbow_Show" for instant demo! 🌈

### **Option 2: Portable Executable**
1. Download `DMXMaster-LTS-1.0.0.exe`
2. Create folders: `data\shows`, `config`, `logs`
3. Run the executable
4. Configure network settings in GUI

### **Option 3: From Source**
```bash
git clone https://github.com/truongcongdinh97/DMX-Master.git
cd DMX-Master
pip install -r requirements.txt
python main.py
```
